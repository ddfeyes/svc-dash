# Session VWAP Band Chart

**Date:** 2026-03-15
**Feature:** VWAP ± 1σ / 2σ shaded band overlay on price

## Problem
Traders need to see not just VWAP but the statistical spread around it — the bands show where price is "expected" to be vs extended moves.

## Data Source
- Existing `trades` table: price, qty, ts
- Compute from session start (last 8h) or configurable window
- For each candle bucket: VWAP (cumulative), σ = sqrt(sum(vol*(price-vwap)²)/sum(vol))

## Endpoint
`GET /api/vwap-band?symbol=X&window=28800&bucket=60`
Returns: [{ts, close, vwap, upper1, lower1, upper2, lower2}]

## Frontend
New card "📐 VWAP Bands" with Chart.js line chart:
- Price line (yellow)
- VWAP line (blue dashed)
- ±1σ band (blue fill 10% opacity)
- ±2σ band (blue fill 5% opacity)

## Approach
Compute incrementally candle-by-candle (cumulative). No new tables needed.
